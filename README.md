# 🌦️ Weather Report Website

## 📌 Overview

This project is a Weather Report Website built using **HTML, CSS, and JavaScript** with a clean and modern user interface. It uses a weather API to fetch real-time data and display accurate weather information based on user input.

## 🚀 Features

* Allows user to enter a **city name**
* Fetches real-time weather data using an API
* Displays:

  * 🌡️ Temperature (in Celsius)
  * 💧 Humidity
  * 🌬️ Wind speed
* Clean, modern, and user-friendly UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## 🎯 Purpose

The purpose of this project is to practice working with APIs and build a real-world interactive web application with modern UI design.

## 📷 UI Highlights

* Minimal and modern design
* Smooth user experience
* Responsive layout

## 📌 How It Works

1. User enters a city name.
2. The application sends a request to the weather API.
3. It fetches real-time weather data.
4. Displays:

   * Temperature (°C)
   * Humidity
   * Wind speed

## 💡 Future Improvements

* Add 5-day or 7-day weather forecast
* Improve UI with animations
* Add location-based weather detection

## 🙌 Conclusion

This project helped in understanding API integration and strengthening JavaScript concepts while building a practical and useful application.
